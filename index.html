Frontend development template

This is a starter file for your own projects.

Advantage of using this template:

  • Start your projects faster
  • Preinstalled plugins for Gulp
  • Normalize.css comes preinstalled
  • Index.html, main.js and app.sass precreated
  • Sass partials created for easier structure
  • Folder structure already created

Features:

  • Convert SASS/SCSS to CSS
  • Add's vendor prefixes automatically
  • Concanate JS
  • Minify JS and CSS
  • Copy assets to build directory
  • Refresh browser while developing

Note: If you have no use normalize.css. Remove include lines from index.html

Please see the GitHub repository for any details and bugs.
